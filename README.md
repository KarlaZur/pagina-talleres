# Sistema de inscripción a talleres

Este proyecto es una página web para el registro de participantes e inscripción a talleres.

## Tecnologías utilizadas

- PHP
- MySQL
- Docker
- HTML
- CSS

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Cierre de sesión
- Visualización de talleres disponibles
- Inscripción a talleres
- Desinscripción de talleres
- Conteo automático de asistentes por taller

## Requisitos

Tener instalado:

- Docker
- Docker Compose
